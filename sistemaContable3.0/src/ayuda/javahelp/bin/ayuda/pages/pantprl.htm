Pantalla Principal

El sistema mostrará el panel Principal con las funciones disponibles para el uso las cuales se muestran a continuación: 

Menú principal.-

Contiene todas las opciones que puede ejecutar el sistema, como son administración de clientes, proveedores, productos, reportes, entre otros.

Barra de herramientas.-

Contiene Accesos directos de las opciones mas comunes del sistema.

Area de Trabajo.-

En esta área se mostrarán todas las funciones que el usuario requiera del sistema a manera de hojas de trabajo.

Identificador de Sesión.-

Muestra por pantalla que usuario esta utilizando el sistema y que privilegios tiene en el mismo.
