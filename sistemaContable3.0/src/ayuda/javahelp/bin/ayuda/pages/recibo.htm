Menú Ventas

Las opciones de este menú le permitirán llevar un control de las ventas de su negocio, controlar los recibos que emite, además de chequear y modificar los mismos de manera amigable.

Además se ha implementado una opción llamada Ordenes de Trabajo las cuales le permitirán organizar en una sola hoja los elementos o productos necesarios para crear un producto que se realiza bajo pedido.

Cobro de Facturas HK (Ninguna)

Para poder registrar la manera como sus clientes han pagado o deberán pagarán los recibos que usted emitió tendra que usar cualquiera de los siguientes búsquedas de recibos de la sección de Consultas .

El sistema permitirá crear recordatorios automáticamente cuando se genere algún pago a plazo, además de almacenar la manera de pago.

En este menú encontrará únicamente que modos de pago acepta el negocio.

Por defecto al almacenar un recibo el sistema lo considera como pagado en "Efectivo" pero esta información puede ser modificada por el usuario.

Ordenes de Trabajo HK (Alt+Ctrl+E)

Las ordenes de trabajo u ODT le permitirá hacer un listado de productos necesarios para conformar un solo producto el cual esta compuesto por otros. Además le permite llevar un control de adelantos que se realizan para iniciar este tipo de trabajos, ademas de permitir la facilidad de asignar el trabajo a un empleado para llevar un control acerca del avance del mismo.

Para poder ingresar datos en las ODT primero debemos registrar datos básicos de los materiales o productos que requiere el producto a elaborar, para lo cual usuamos este formulario llamado Tipo de Trabajo.

Algo importante de mencionar es que este módulo de ODT trabaja por medidas, por lo cual el campo de precio mínimo es el costo aproximado de un trabajo que no requiere muchos materiales o que no cumple con una medida mínima para elaborarlo .

un ejemplo: un producto "X" de medidas 1x0.50 costará lo que anotamos como " precio mínimo" mientras algo que sobrepase o sea igual a 1x1 en medidas llevara su proporcional según el campo "precio".

Recibo Venta HK (Alt+Ctrl+R)

Las ordenes de trabajo u ODT le permitirá hacer un listado de productos necesarios para conformar un solo producto el cual esta compuesto por otros, ademas le permite llevar un control de adelantos que se realizan para iniciar este tipo de trabajos, ademas de permitir la facilidad de asignar el trabajo a un empleado para llevar un control acerca del avance del mismo.

Para poder ingresar datos en las ODT primero debemos registrar datos básicos de los materiales o productos que requiere el producto a elaborar, para lo cual usuamos este formulario llamado Tipo de Trabajo.

El area de orden de trabajo permite anadir al recibo los datos de "una" orden de trabajo para ser facturados .

Es importante antes de empezar a ingresar datos en el recibo especificar si es una factura o una nota de venta, en la sección "Escoja una opción".

El ícono junto a los datos del cliente nos permite buscar un registro existente.

Recuerde se puede añadir en la lista del recibo hasta 10 item diferentes.

La opción ver reporte permitirá al usuario observar la factura generada en una versión que puede ser exportada en varios formatos.

La opción imprimir nos permitirá, realizar la acción que tiene etiquetado sobre algún recibo en un impresora.

La sección "Aplica cobro del I.V.A. puede ser utilizada si usted requiere imprimir el recibo grabado con el impuesto "0%".

Cuando se registra una factura, esta se alamcena en el sistema como pagado en efectivo, esto se lo puede cambiar realizar la búsqueda.
